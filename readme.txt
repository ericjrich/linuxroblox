To Install Roblox On Linux:
  *Goto The Folder That Represents Your Distro
  *Download The "e-installroblox.eb" script
  *Put it somewhere and open a terminal there
  *Run: chmod +x e-installroblox.eb
  *Run: ./e-installroblox.eb
  *WATCH For Instructions in the terminal window, so that you know what to do when the GUI stuff happens
